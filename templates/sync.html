Setup Required

To sync with Google Contacts, you need to set up API credentials:

  1. Go to the Google Cloud Console
  2. Create a project (or select existing) and enable the People API
  3. Go to Credentials → Create Credentials → OAuth 2.0 Client ID
  4. Set application type to Desktop app
  5. Download the JSON and save it as:
    ~/.config/contact-list/credentials.json
